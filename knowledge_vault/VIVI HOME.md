## Objectif MVP

VIVI est d’abord un assistant local de discussion.

Je lui parle, elle me répond.

Le MVP doit permettre :

- interface dédiée simple ;
- LM Studio comme provider local ;
- chat local ;
- RAG Obsidian ;
- sources visibles ;
- runtime status ;
- protection simple.
