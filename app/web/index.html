Mémoire VIVI

Créer une note candidate dans Obsidian Inbox, à relire avant validation.

La note restera en brouillon, non indexée, avec relecture humaine requise.

Utiliser VIVI
  • Envoyer un message : VIVI utilise automatiquement le vault Obsidian pour enrichir sa réponse.
  • Sources : quand VIVI trouve des notes pertinentes, un encart Sources apparaît. Clique pour les voir.
  • LM Studio : doit être lancé avec un modèle chargé.
  • Si le modèle est indisponible : vérifier LM Studio, le modèle chargé et la configuration .env.
